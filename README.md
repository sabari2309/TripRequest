# TripRequest

Problem Statement
You are required to develop a system to manage trips and their associated services. The system should allow users to create a trip, add services to the trip, and finally request the trip, which will display the requested services as the finalized trip tickets/order. The process is as follows:
Create a Trip:
•	Input: Trip name and travel date.
•	Output: A new trip is created with the provided name and travel date.
 

Add Services to the Trip:
•	Input: Services to be added to the trip.
•	Output: The specified services are added to the trip.

 
Request the Trip:
•	Input: Trip request.
•	Output: Display the requested services as the finalized trip tickets/order.
Detailed Steps
Create a Trip:
•	Prompt the user to input the trip name and travel date.
•	Store the trip details (name and travel date).
Add Services to the Trip:
•	Allow the user to add one or more services to the created trip.
•	Store each service in association with the trip.
Request the Trip:
•	When the user requests the trip, retrieve all services associated with the trip.
•	Display the trip details along with the associated services as the finalized trip tickets/order.
